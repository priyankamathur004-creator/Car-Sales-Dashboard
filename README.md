# Car-Sales-Dashboard

# Project Overview

This project presents an interactive Car Sales Dashboard that visualizes sales data from 1984 to 2022. The dashboard highlights trends in car prices, mileage, manufacturers, and models, providing actionable insights for automotive sales analysis.

# Dashboard Highlights

Key Metrics
13.83K – Average Price

112.52K – Average Mileage

15 – Types of Models

# Visualizations

Model by Manufacturer – Bar chart showing manufacturers (Ford, VW, Toyota, BMW, Porsche) and their model counts.

Manufacturer by Model – Pie chart showing distribution of popular models (Mondeo, Golf, Polo, Focus, Fiesta, Passat, RAV4).

Price & Mileage by Fuel Type – Scatter plot comparing price vs. mileage across fuel types.

Price Trends by Year (1980–2022) – Line chart showing long‑term price evolution.

Yearly Price Distribution – Bar chart showing yearly breakdown of car prices.

# Methodology
Data Collection – Historical car sales dataset (1984–2022).

Data Cleaning – Handled missing values, standardized metrics.

Visualization Tools – Built using Power BI 

Analysis Focus – Price trends, mileage impact, manufacturer/model distribution, fuel type comparison.

# Insights
Car prices increased steadily until ~2020, followed by a decline.

Average mileage remains high, indicating used car dominance.

VW and Ford dominate model counts, with Golf and Polo among the most popular.

Fuel type significantly influences both price and mileage.

# Conclusion

This dashboard provides a comprehensive view of car sales trends, helping stakeholders understand pricing, mileage, and manufacturer performance over time.
